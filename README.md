# Генератор документов с печатью

Простой локальный сайт: заполняете форму — получаете готовый PDF с наложенной печатью и подписью.

## Как запустить

Откройте файл `index.html` в браузере (Chrome, Edge, Firefox) — двойным щелчком или через
«Открыть с помощью → браузер». Подключение к интернету нужно один раз при загрузке страницы
(сайт подгружает библиотеки jsPDF и html2canvas с CDN), дальше всё работает локально, никакие
данные никуда не отправляются.

## Как пользоваться

1. Выберите тип документа (сейчас доступен один — перевод свидетельства о рождении на немецкий).
2. В разделе «Печать и подпись» выберите PNG-файлы печати и подписи. Удобно заранее положить их
   в папку `assets` рядом с сайтом — она уже создана, инструкция там же (`assets/README.txt`).
3. Заполните поля документа.
4. Нажмите «Сгенерировать PDF» — браузер сам соберёт документ и сразу скачает готовый PDF-файл.

Все введённые данные и выбранные изображения сайт запоминает в браузере (localStorage), поэтому
при следующем открытии форма будет уже заполнена.

## Как добавить новый тип документа

Файлы устроены так, чтобы расширение не требовало переписывать всё с нуля:

- **`templates.js`** — описание полей формы для каждого документа (название, подписи полей,
  группировка по разделам). Добавьте новый объект в массив `TEMPLATES`.
- **`pdf-builder.js`** — макет документа в виде HTML/CSS-разметки (`DOCUMENT_RENDERERS`).
  Добавьте функцию-рендерер с тем же `id`, что и в `templates.js`, которая возвращает HTML
  будущего документа, заполненный значениями из формы.

Сайт сам отрисует форму по описанию из `templates.js`, а готовый HTML конвертирует в PDF —
писать код генерации PDF заново не нужно.

## Технические детали (на будущее)

- Текст рендерится силами браузера (через `html2canvas`), поэтому кириллица, немецкие умляуты
  и любые другие символы отображаются корректно — без проблем со встроенными шрифтами PDF.
- Изображения печати/подписи хранятся как base64 в localStorage браузера — никуда не отправляются.
- Если документ окажется длиннее одной страницы A4, генератор автоматически разобьёт его на
  несколько страниц PDF.
